    Research

    My research interests include network security, uncertainty quantification, deep reinforcement learning, and IoT applications. I'm currently working on projects related to moving target defense, smart farm monitoring systems, and in-vehicle network security. Representative papers are highlighted.

    		PPO-UE: Proximal Policy Optimization via Uncertainty-Aware Exploration
    Qisheng Zhang, Zhen Guo, Audun Jøsang, Lance Kaplan, Feng Chen, Dong Hyun Jeong, Jin-Hee Cho,
    AAAI (workshop), 2023
    paper

    We propose PPO-UE, a PPO variant equipped with self-adaptive uncertainty-aware explorations (UEs) based on a ratio uncertainty level. The proposed PPO-UE is designed to improve convergence speed and performance with an optimized ratio uncertainty level.
    		Uncertainty-Aware Reward-based Deep Reinforcement Learning for Intent Analysis of Social Media Information
    Zhen Guo, Qi Zhang, Xinwei An, Qisheng Zhang, Audun Jøsang, Lance Kaplan, Feng Chen, Dong Hyun Jeong, Jin-Hee Cho,
    AAAI (workshop), 2023
    paper

    We propose an intent classification framework that can best identify the correct intent of fake news. We use an uncertainty-aware subjective logic based immediate reward in our DRL frameworks.

    		An Attack Resilient and Energy-Adaptive Monitoring System for Smart Farms
    Qisheng Zhang, Yash Mahajan, Ing-Ray Chen, Dong Sam Ha, Jin-Hee Cho,
    GLOBECOM, 2022
    paper / code / slides

    In this work, we use deep reinforcement learning and subjective logic to propose an energy-adaptive monitoring system for a solar sensor-based smart animal farm (e.g., cattle).
    		Network Resilience Under Epidemic Attacks: Deep Reinforcement Learning Network Topology Adaptations
    Qisheng Zhang, Jin-Hee Cho, Terrence J. Moore,
    GLOBECOM, 2021
    paper / code / slides

    We propose a Deep reinforcement learning (DRL)-based NETwork Adaptations for network Resilience algorithm, namely DeepNETAR, which aims to generate robust network topologies against epidemic attacks under different system objectives.

    		DREVAN: Deep Reinforcement Learning-based Vulnerability-Aware Network Adaptations for Resilient Networks
    Qisheng Zhang, Jin-Hee Cho, Terrence J. Moore, Frederica Free Nelson,
    CNS, 2021
    paper / code / slides

    We propose a vulnerability-aware network adaptation framework that can generate a robust network topology against epidemic attacks by leveraging deep reinforcement learning (DRL) to build a resilient network.
    		Diversity-By-Design for Dependable and Secure Cyber-Physical Systems: A Survey
    Qisheng Zhang, Abdullah Zubair Mohammed, Zelin Wan, Jin-Hee Cho, Terrence J. Moore,
    TNSM
    paper

    In this work, we are particularly interested in providing an in-depth, comprehensive survey of existing diversity-based approaches, their insights, and associated future work directions for building secure and dependable CPSs.
    		Vulnerability-Aware Resilient Networks: Software Diversity-based Network Adaptation
    Qisheng Zhang, Jin-Hee Cho, Terrence J. Moore, Ing-Ray Chen,
    TNSM
    paper

    By leveraging the principle of software polyculture to ensure security in a network, we propose a vulnerability based software diversity metric to determine how a network topology can be adapted to minimize security vulnerability while maintaining maximum network connectivity.
